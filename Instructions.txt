1. Build dash application and make sure these 2 lines are at the top

app = dash.Dash(__name__)
server = app.server

2. Get a domain

Buy a domain at namecheap.com or at godaddy.com

3. Point domain name to EC2 instances
for godaddy.com, go to DNS manage zone --> enter your ec2 elastic IP in the first row
where " A  @  value ". Replace the value with your ec2 elastic IP

4. Nginx and gunicorn

sudo apt-get install nginx
pip install gunicorn

sudo /etc/init.d/nginx
sudo rm /etc/nginx/sites-enabled/default
sudo touch /etc/nginx/sites-available/application
sudo ln -s /etc/nginx/sites-available/application /etc/nginx/sites-enabled/application
sudo nano /etc/nginx/sites-enabled/application

And paste (exactly):

server {
    location / {
        proxy_pass http://localhost:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

5. Run your app
Navigate to your app folder, and then type
$ gunicorn myapp:app.server
(if your app name is myapp.py)
To use port 80, specify in the last line of your dash app like follow:
if __name__ == '__main__':
    app.run_server(debug=True, host='0.0.0.0', port=80)
    
To run in the background: GeneMiner/flask-dash-app $ gunicorn app:app -D

Kill the process: pkill gunicorn
